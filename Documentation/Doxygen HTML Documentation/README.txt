GNATS uses Doxygen to generate HTML document.

Software: Doxygen 1.8.15(provided in the directory)


Description of Files/Directories
================================
html            <DIR>       Generated HTML files after running Doxygen
src             <DIR>       Pages and materials to generate documents
nats_client_doxygen.conf    Doxygen configuration file


Installation of Doxygen
=======================
Please unzip the software and follow the instruction


Generate Document
=================
# Execute command
./run_doxygen.sh
